# 集成化mod管理器 v3.1.0

> Non-official notice:
> This project is an unofficial fan-made tool. It is not affiliated with, endorsed by, or sponsored by XXMI, any game publisher, or any related developers.

这是一个运行在 Windows 上的 WinUI 3 图形工具，用于管理多仓库 Mod 工作区、浏览两层 Mod 文件夹、导入压缩包、记录每个 Mod 的快捷键说明、设置预览图、保存 Mod 链接，并浏览在线 Mod。

当前版本：

- 显示版本：`v3.1.0`
- 文件版本：`3.1.0.0`

## 主要功能

- 支持多仓库工作区，可分别配置 Mod 存储路径、目标路径和启动器路径
- 支持第一层分类与第二层 Mod 文件夹浏览
- 支持复制、删除、预览图、快捷键说明和 Mod 链接管理
- 支持导入 `.zip`、`.zipx`、`.7z`、`.rar`、`.tar`、`.tar.gz`、`.tar.bz2`、`.tar.xz`、`.tar.zst`、`.gz`、`.tgz`、`.bz2`、`.xz`、`.zst`、`.cab`
- 支持在线 Mod 浏览、筛选、分页、详情预览、下载与解压
- 支持按仓库分别配置 GameBanana 来源、目标游戏和皮肤分类 ID
- 支持中英切换和深浅色主题

## v3.1.0 更新

- 新增按仓库配置在线分类的能力，不同游戏可分别填写不同的 GameBanana 皮肤分类 ID
- 放宽在线分类过滤规则，只要分类名称中包含 `Skins` 就会显示
- 显示版本更新为 `v3.1.0`，文件版本更新为 `3.1.0.0`
