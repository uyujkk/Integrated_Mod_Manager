# 更新报告

## 项目

集成化mod管理器 WinUI 3 版

## 当前版本

- 显示版本：`v3.1.0`
- 文件版本：`3.1.0.0`

## 本次更新内容

### 1. 在线分类配置改进

- 新增按仓库配置在线分类的能力
- 可分别填写在线来源、目标游戏和 GameBanana 皮肤分类 ID
- 不同游戏现在可以使用不同的在线分类配置

### 2. 在线分类过滤优化

- 不再要求分类名称必须完全等于 `Skins`
- 只要分类名称中包含 `Skins` 关键词，就会显示在在线列表中
- 更适合不同游戏中存在分类名称变体的情况

### 3. 版本同步

- 应用显示版本更新为 `v3.1.0`
- 文件版本更新为 `3.1.0.0`
- 同步更新 README、CHANGELOG 和使用说明
