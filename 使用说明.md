# 集成化mod管理器 v3.0.1 使用说明

## 1. 运行方式

1. 将发布压缩包完整解压到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后，先配置：
   - `Mod 仓库`
   - `目标文件夹`
   - `启动器`（可选）
4. 点击 `刷新目录`

## 2. 推荐目录结构

程序按两层目录工作，推荐结构如下：

```text
Mod 仓库
├─ 分类A
│  ├─ Mod1
│  ├─ Mod2
│  └─ Mod3
├─ 分类B
│  ├─ Mod4
│  └─ Mod5
└─ 分类C
   └─ Mod6
```

- 第一层：分类文件夹
- 第二层：实际操作的 Mod 文件夹

## 3. 基本使用流程

1. 选择 `Mod 仓库`
2. 选择 `目标文件夹`
3. 点击 `刷新目录`
4. 在左侧选择第一层分类
5. 在中间选择第二层 Mod
6. 根据需要复制、删除、导入压缩包、设置预览图、记录快捷键或打开链接

## 4. 压缩包导入

- 顶部导入按钮会把压缩包导入到当前选中的第一层文件夹
- 目前支持：`.zip`、`.7z`、`.tar`、`.gz`、`.tgz`、`.bz2`、`.xz`
- 第二层区域也支持拖拽压缩包导入

## 5. 预览图

程序会优先读取以下命名的图片：

- `preview.*`
- `cover.*`
- `thumbnail.*`
- `image.*`

如果没有这些文件，会尝试显示当前 Mod 文件夹中的第一张图片。

你也可以直接把图片拖到预览区域，程序会自动将其设置为当前 Mod 的预览图。

## 6. 快捷键与描述

每个第二层 Mod 都可以单独保存自己的快捷键和描述。

支持：

- 单个按键，例如 `1`、`Q`、`F1`
- 组合按键，例如 `Ctrl+1`
- 符号按键，例如 `/`、`;`、`[`、`]`、`\`

## 7. Mod 链接

每个 Mod 都可以绑定一个独立网页链接。

- 链接会显示在预览区域下方
- 点击快速访问按钮后，会调用系统默认浏览器打开

## 8. 在线 Mod

在线页面支持：

- 浏览 GameBanana 条目
- 搜索、筛选与分页
- 查看详情和预览图
- 下载并解压到当前仓库
- 记录来源、远程 ID 和更新时间

## 9. 启动器

如果配置了启动器路径，点击运行按钮后会直接启动对应程序。
