# 集成化mod管理器 v3.1.0 使用说明

## 1. 运行方式

1. 将发布压缩包完整解压到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后先配置：
   - `Mod 仓库`
   - `目标文件夹`
   - `启动器`（可选）
4. 点击 `刷新目录`

## 2. 压缩包导入

- 顶部导入按钮会把压缩包导入到当前选中的第一层文件夹
- 第二层区域也支持拖拽压缩包导入
- 当前支持：`.zip`、`.zipx`、`.7z`、`.rar`、`.tar`、`.tar.gz`、`.tar.bz2`、`.tar.xz`、`.tar.zst`、`.gz`、`.tgz`、`.bz2`、`.xz`、`.zst`、`.cab`

说明：

- `.zip` 使用程序内置解压
- `.tar`、`.gz`、`.tgz`、`.bz2`、`.xz`、`.zst` 及对应 `.tar.*` 使用系统 `tar.exe`
- `.7z`、`.rar`、`.zipx`、`.cab` 使用 7-Zip 解压

## 3. 在线 Mod

- 在线页会根据当前仓库配置读取 GameBanana 条目
- 现在支持按仓库分别配置：
  - 在线来源
  - 目标游戏
  - 皮肤分类 ID
- 不同游戏可以建立不同仓库，分别填写对应的分类 ID
- 在线分类过滤规则已放宽，只要分类名称中包含 `Skins` 就会显示

## 4. 快捷键与预览

- 每个第二层 Mod 都可保存自己的快捷键说明和功能描述
- 支持预览图显示与图片拖拽导入
- 支持为每个 Mod 保存网页链接并快速打开

## 5. 当前版本

- 显示版本：`v3.1.0`
- 文件版本：`3.1.0.0`
