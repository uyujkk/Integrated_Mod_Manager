# 集成化mod管理器 v3.0.2 使用说明

## 1. 运行方式

1. 将发布压缩包完整解压到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后，先配置：
   - `Mod 仓库`
   - `目标文件夹`
   - `启动器`（可选）
4. 点击 `刷新目录`

## 2. 压缩包导入

- 顶部导入按钮会把压缩包导入到当前选中的第一层文件夹
- 第二层区域也支持拖拽压缩包导入
- 目前支持：`.zip`、`.zipx`、`.7z`、`.rar`、`.tar`、`.tar.gz`、`.tar.bz2`、`.tar.xz`、`.tar.zst`、`.gz`、`.tgz`、`.bz2`、`.xz`、`.zst`、`.cab`

说明：

- `.zip` 使用程序内置解压
- `.tar`、`.gz`、`.tgz`、`.bz2`、`.xz`、`.zst` 及对应的 `.tar.*` 格式使用系统 `tar.exe`
- `.7z`、`.rar`、`.zipx`、`.cab` 使用 7-Zip 解压
- 发布包会优先使用随包提供的 `Tools\\7z.exe`
